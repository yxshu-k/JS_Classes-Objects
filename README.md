# JS Classes & Objects 

## Description

This project demonstrates Object-Oriented Programming (OOP) concepts in JavaScript, focusing on **Classes and Objects**. It implements a practical college website system with User and Admin role management.

## Project Includes

- **User Class**: Base class with name, email properties and `viewData()` method
- **Admin Class**: Inherits from User, includes `editData()` method for website data modification
- **Practical Implementation**: Real-world example of user role hierarchies
- **HTML Entry Point**: Browser-based execution support
- **Console Logging**: View data operations in browser console or Node.js terminal

## Project Structure

```
JS_Classes_Obj/
├── README.md          # Project documentation
├── index.html         # HTML entry point
└── PracticeQsn.js     # Main JavaScript file with class definitions
```

## Installation & Setup

No installation required. This is a vanilla JavaScript project with no external dependencies.

## Usage

### Run in Browser
1. Open `index.html` in your web browser
2. Open Developer Console (F12 or Right-click → Inspect → Console)
3. View the output of User and Admin class operations

### Run with Node.js
```bash
node PracticeQsn.js
```

## Features

- **Class Definition**: User and Admin classes with constructors
- **Inheritance**: Admin class extends User class using `extends` keyword
- **Methods**: View data (User) and edit data (Admin) functionality
- **Instance Creation**: Multiple user instances with different roles

## Technologies Used

- JavaScript (ES6+)
- HTML5
- Node.js (optional)

## Code Concepts

- Class declaration and constructor
- Object instantiation with `new` keyword
- Class inheritance with `extends` and `super()`
- Instance methods and properties
- `this` keyword binding
